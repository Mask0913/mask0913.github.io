#C语言学习：

##指针：

导语：在编程语言中，当我们做程序的时候回遇到一些麻烦： 当想定义一个函数来修改main函数中的值的时候，发现函数无法修改main中的变量，那么，有没有一种办法来解决呢。

我们都知道，每个变量中都有自己对应的内存地址，那么，如果在函数中直接修改内存地址的值，主函数的值是不是就修改了呢？

答案明显是肯定的，因为变量储存的地址就是变量的位置，修改这个地址中的值，变量自然也会改变。

那么，C语言中是如何实现这种修改内存地址中的值的办法呢？

这就引入了新的知识，指针。

###如何使用指针：

指针，就是利用地址，它的值直接指向（points to）存在电脑存储器中另一个地方的值。
