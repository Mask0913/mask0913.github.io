#第一次自主写个小爬虫的记录

题外话：这是寒假老师强迫我们早期，必须在微信里面发早，闲来无事，就写个脚本统计下每个人的签到状况。

具体要求：

从一个混乱的微信聊天记录中获取每个人发“早”的次数

微信聊天记录：

可以看出，图片中的聊天记录相当混乱，想要获取指定的消息有一点难。

通过观察数据，我么可以看出每一个人的微信名字长度都是50个以内，所以，我们的初步提纯正则表达式为：

name_expression = '[\s\S]{3,50}\n\n 早'

这样，我们就能提取出早前面的一些字符了

然后 拿到提取后的数据是这样的：

上面不仅带着日期 还带着\n和早，这明显不能作为我们的结果来使用，于是我们接着来提纯：

next1_exprission = '\n([\s\S]*?)\n\n 早'

这次的结果是这样的：

这次的结果明显清楚的能看到人了，但是这还不够，因为还有日期干扰，我们最后还要做统计，这样肯定不行。

于是，我们写下剔除日期的正则表达式：

next2_exprission = '[\s\S]{3,50}\)'

剔除后我们拿到结果。
